تبدیل ریال(تومان) به Rial(Toman) در افزونه WPML برای ووکامرس.

== Description ==

اگر شما یک سایت دوزبانه دارید و می‌خواهید در حالت انگلیسی برای یک محصول ووکامرس به جای کلمه ریال یا تومان کلمه Rial یا Toman نمایش داده شود این افزونه می‌تواند به شما کمک کند.
